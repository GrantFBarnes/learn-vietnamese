You will need to have the Vietnamese language/keyboard installed in order to follow these instructions.

Vietnamese Alphabet

a, ă, â, b, c, d, đ, e, ê, g, h, i, k, l, m, n, o, ô, ơ, p, q, r, s, t, u, ư, v, x, y

Typing â, ê, ô, đ

These each have the "hat", and these can be typed by typing the letter twice in a row.

  • type "aa" for â
  • type "ee" for ê
  • type "oo" for ô
  • type "dd" for đ

Typing ư, ơ and ă

Two of these have the "horn", and the one has the "bowl". They are all typed the same way by appending w to the letter.

  • type "uw" or "w" for ư
  • type "ow" for ơ
  • type "aw" for ă

The Five Accent Marks/Diacritics

The following marks are able to be applied to any vowels (including the ones above that already have marks).

Name Mark Append Example With a
acute ´ s type "as" for á
grave ` f type "af" for à
hook ̉ r type "ar" for ả
tilde ~ x type "ax" for ã
dot . j type "aj" for ạ

Typing English Names

If you want to type an english name such as Mary or Lawrence, you will need to do something to void typing Vietnamese characters. You notice the "ar" in Mary will produce ả, and the "awr" in Lawrence will produce ẳ. In order to stop the special vietnamese characters while the keyboard is enabled you can append that special character twice.

  • type "arr" for ar
  • type "awwr" for awr
